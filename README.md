# BatchGemist
BatchGemist was een batchscript om video's van verscheidene websites te downloaden, of de videolink ervan te achterhalen.

Dit was voor mij een grote leerschool om ten eerste Xidel onder de knie te krijgen én natuurlijk Batch zelf met al z'n tekortkomingen.
Begin 2017 heb ik met mijn andere [project](https://github.com/Reino17/ffmpeg-windows-build-helpers) (als Windows gebruiker) flink wat ervaring opgedaan met Bash (Linux). Eind 2018 ben ik daarom eens gaan kijken of ik BatchGemist ook in Bash werkend kon krijgen. Ik ben nog steeds een Windows gebruiker, maar m.b.v Cygwin is dit zeker gelukt, met [BashGemist](https://github.com/Reino17/bashgemist/) als resultaat. Geen vervelende tekortkomingen meer die omzeilt moeten worden én de Xidel queries zijn nu goed leesbaar zonder verplichte escapecharacters.

Ik ga mij volledig richten op BashGemist. De ontwikkeling van BatchGemist zal daardoor ten einde komen.
